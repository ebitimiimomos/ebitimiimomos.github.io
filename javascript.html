Project Overview

In this project, I created a simple web-based game called "Game of Dice." The game involves rolling dice and calculating points based on specific rules. Here is a summary of what I did:

1. Setup Stage

In the setup stage, I designed a user interface with an input field where the user can enter the number of dice (between 3 and 6) to play with. When the user clicks the "Start Game" button, I parsed the input to ensure it is a valid natural number within the specified range. If the input is valid, I hid the setup section, displayed the play section, and set the round number to 1.

2. Play Stage

In the play stage, I implemented the logic for rolling the dice and calculating points won for each round. I used random number generation to simulate dice rolls. The number of dice rolled is based on the user's input from the setup stage. After calculating the points won, I updated the balance, total points earned, and number of rounds played. The dice values and points won are displayed on the interface.

3. End Stage

When the user decides to end the game by clicking the "End Game" button, I hid the play section and displayed the end section. In the end stage, I provided a summary of the game's results, including the total number of rounds played, the total points earned, and the average points earned per round. I also created a bar graph using canvas to visualize the points earned in each round.

4. Game Instructions

I included instructions for playing the game, guiding the users through the setup, play, and end stages. I explained the rules for calculating points based on different dice combinations.

Conclusion

Overall, this project provides an interactive and enjoyable game experience for users, allowing them to play rounds of dice and see their performance results at the end of the game.
