Task 2 involved implementing five different voting systems: dictatorship, scoring rule, plurality, veto, borda, and harmonic. The `dictatorship` function allows a dictator agent to make a decision, and the `scoringRule` function calculates the winner based on agents' scores for alternatives.

The `plurality` function determines the winner by selecting the alternative with the most first-place votes. The `veto` function ranks alternatives based on the number of times they are ranked higher than others, and the `borda` function assigns scores to alternatives based on their ranks and selects the one with the highest score.

The `harmonic` function calculates the scores for each alternative based on the harmonic mean of their ranks and selects the one with the highest score. In all these functions, tie-breaking is implemented using the `tieBreaker` function, which resolves ties based on user-defined rules.
