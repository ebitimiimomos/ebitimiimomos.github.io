Project Overview

In this project, I designed and implemented a web application for booking IT training sessions at the University. The application allows users to select a topic, view available training sessions for that topic, and book a session by providing their name and email address. The project utilized a combination of HTML, CSS, and PHP to create the web pages and interact with the database.

Database Setup

<?php
              // Database connection details
              $hostname = 'your_hostname';
              $dbname = 'your_database_name';
              $username = 'your_username';
              $password = 'your_password';
              $charset = 'utf8mb4';
              // Create a PDO object for secure and efficient database interaction
              try {
              $dsn = "mysql:host=$hostname;dbname=$dbname;charset=$charset";
              $pdo = new PDO($dsn, $username, $password);
              $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
              } catch (PDOException $e) {
              echo "Connection failed: " . $e->getMessage();
              exit;
              }
              ?>

Topic Selection and Session Booking

The main web page consists of two sections: selecting a topic and booking a session. Initially, users are presented with a dropdown menu containing distinct topics with available capacity. The topics are retrieved from the database and ordered alphabetically. Users can choose a topic and submit their selection.

Confirmation Display

After successful booking, a confirmation message is displayed to the user, showing their name, email, selected topic, and session time. Users have the option to make another booking by clicking the "Make Another Booking" button.

Error Handling

Throughout the code, I implemented error handling to catch exceptions and display appropriate error messages if any database or input validation issues arise.

Conclusion

This web application provides a user-friendly and secure platform for students to book IT training sessions at the University. It leverages PHP's capabilities to interact with the database and ensures data integrity and user validation for a smooth booking experience.
