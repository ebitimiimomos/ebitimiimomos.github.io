Solving Semantic Segmentation Task Using Deep Learning Models

Python + Ms Excel + Ms Word
View Code

In this project, I worked on performing semantic segmentation using deep learning models with the Cambridge Labelled Objects in Video dataset. The goal was to develop and compare different models for accurately segmenting objects in images.

Dataset

The dataset contains original images and labeled frames in uncompressed 24-bit color PNG format. It includes 101 images with each pixel manually assigned to one of 32 object classes relevant in a driving environment.

Models Implemented

I implemented three different models for semantic segmentation: FCN, UNet, and DeepLab (DeepLabV3). Each model utilizes different architectures and techniques to extract meaningful features from input images and generate accurate segmentation maps.

  • The UNet model consists of an encoder network, a bottleneck layer, and a decoder network with skip connections.
  • The DeepLab model is based on the ResNet-50 architecture, with a base feature extraction module and a classifier module.
  • The FCN model follows an encoder-decoder structure to combine features from different resolutions for generating the final segmentation map.

Techniques Used

To address overfitting, I incorporated techniques such as dropout and weight decay into the models. Data augmentation techniques like rotation, scaling, and flipping were used to increase the diversity of the training data and reduce overfitting.

Hyperparameters and Evaluation

The hyperparameters and techniques were systematically tuned to achieve optimal performance for each model. The learning rate was set at 0.001, and the number of epochs was chosen as five. The models were evaluated using metrics like Intersection over Union (IoU) and pixel accuracy.

Results

The results showed that all three models performed well, with DeepLab achieving the highest IoU and pixel accuracy. The choice of model depended on specific requirements, with DeepLab being preferred for precise object boundaries and FCN for computational efficiency and real-time performance.

Advantages and Disadvantages

However, each model had its own advantages and disadvantages. The UNet model combined low-level and high-level features but required more computational resources. The DeepLab model utilized hierarchical features and was lightweight, while the FCN model was simple and effective but struggled with fine details and boundaries.

Conclusion

In conclusion, this project provided valuable insights into image segmentation models and their performance characteristics. Challenges were faced in terms of memory requirements and boundary ambiguity. Future improvements could involve exploring different model architectures, hyperparameter tuning, and optimization algorithms. Overall, this project deepened understanding and hands-on experience in developing segmentation models.
